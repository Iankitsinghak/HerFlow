# **App Name**: Woomania

## Core Features:

- User Authentication: Secure user registration, login, and logout using Firebase Authentication with email/password.
- Profile Creation: Allow users to create and manage their profiles, stored in Firestore.
- Cycle Logs CRUD: Enable users to create, read, update, and delete (CRUD) their personal cycle logs using Firestore.
- Blog Posts CRUD: Enable users to create, read, update, and delete blog posts related to women's health, using Firestore.
- Community Posts CRUD: Enable users to create, read, update, and delete community forum posts using Firestore.
- Doctor Questions CRUD: Allow users to ask questions to doctors, which are stored and managed in Firestore.
- Image Upload: Allow users to upload images using Firebase Storage for profile pictures and blog posts.

## Style Guidelines:

- Primary color: Deep magenta (#C2185B) to evoke femininity and strength.
- Background color: Very light pink (#FCE4EC), a heavily desaturated version of the primary.
- Accent color: Purple (#9C27B0), analogous to the primary, adding contrast and visual interest.
- Body and headline font: 'Alegreya', a humanist serif for an elegant, intellectual, contemporary feel.
- Use simple, modern icons related to women's health and community.
- Clean and responsive layout with clear sections for easy navigation.
- Subtle animations for page transitions and interactive elements to enhance user experience.
